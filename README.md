## Installation

Install my-project with Git

```bash
  git clone https://github.com/CainMcRolan/Sol-And-Luna-Dashboard.git
  cd Sol-And-Luna-Dashboard

```

Install dependencies with Composer

```bash
  composer install
```

Add an .env file

```bash
  DATABASE_HOSTNAME=''
  DATABASE_USERNAME=''
  DATABASE_PASSWORD=''
  DATABASE_PORT=''
  DATABASE_NAME=''
  DATABASE_CHARSET=utf8mb4

  PUSHER_AUTH_KEY=''
  PUSHER_SECRET=''
  PUSHER_APP=''
  PUSHER_CLUSTER=''
```
